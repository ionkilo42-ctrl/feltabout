# Feltabout v0.1.0-alpha

Proposed pre-release notes for the first serious open-source alpha snapshot.

## Summary

`v0.1.0-alpha` marks Feltabout as a safety-first, non-clinical open-source foundation for reflection and conversation preparation. This release centers the active MVP 1 stack, clarifies public project boundaries, and improves contributor-facing documentation.

## Highlights

- documented Feltabout as communication-prep and reflection tooling, not therapy
- clarified active vs legacy repository paths
- documented architecture, roadmap, safety boundaries, and OSS application context
- added governance files for contributing, security reporting, licensing, and community conduct
- added GitHub issue templates and a PR template
- documented exact baseline install, run, build, and test commands already supported by the repository

## Active Stack in This Alpha

- `services/api/` FastAPI backend
- `frontend/` Next.js web app
- `apps/mobile/` Expo mobile prototype
- `packages/shared/` shared TypeScript package

## Known Limitations

- pre-release software with mixed active and legacy/reference code
- no claim of therapy, diagnosis, or crisis support capability
- experimental surfaces remain in the repo and should not be mistaken for stable production features
- screenshots and polished demo assets still need to be added

## Suggested Tag Message

`v0.1.0-alpha: safety-first OSS baseline for reflection and conversation prep`
